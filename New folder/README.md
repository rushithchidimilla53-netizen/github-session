# 🌱 Eco-Friendly Packing Store Using AI

A responsive front-end web application for an **Eco-Friendly Packing Store** built using **HTML, CSS, and JavaScript**. The project provides a simple e-commerce experience with user authentication, product browsing, shopping cart, checkout, and payment options.

---

## 📌 Features

- 🔐 User Login & Signup
- 🛍️ Product Listing
- 🔍 Search Products
- ⭐ Product Ratings
- 🛒 Add to Cart
- ➕ Increase/Decrease Product Quantity
- 💰 Automatic Total Price Calculation
- 🏠 Checkout with Address Form
- 💳 Multiple Payment Methods
  - Cash on Delivery (COD)
  - UPI
  - Card Payment
- 📦 Order Storage using Local Storage
- 🔊 Button Click Sound
- 📱 Mobile Friendly Design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## 🚀 How to Run

1. Download or Clone this repository.

```bash
git clone https://github.com/yourusername/eco-friendly-packing-store.git
```

2. Open the project folder.

3. Double-click **ecofreindly.html**

OR

Open it using **VS Code** and install the **Live Server** extension.

Right Click → **Open with Live Server**
